Introduction to Alkanes

Alkanes are the simplest hydrocarbons. They are saturated because they contain only single bonds.

Example: Methane (CH₄), Ethane (C₂H₆), Propane (C₃H₈).

General Formula

The general formula for alkanes is CₙH₂ₙ₊₂.

Example: If n = 4 → C₄H₁₀ (Butane).

Naming Alkanes

Use the root name + “-ane”. E.g., 1 carbon = methane, 2 = ethane, 3 = propane.

For branches: find the longest chain, number the carbon atoms, name the branches.

Isomerism

Isomers have the same formula but different structures. Example: Butane → n-butane & isobutane.

Physical Properties

Boiling point increases with more carbons. Alkanes are non-polar and don’t mix with water.

Combustion

Complete: Alkane + O₂ → CO₂ + H₂O.

Incomplete: Produces CO or soot.

Uses

Used as fuels: cooking gas, petrol, candles.

Environmental Impact

Burning alkanes produces CO₂ → global warming. Incomplete combustion → CO poisoning.
